Descripción
-----------
Este repositorio contiene análisis de datos reproducibles en Python: notebooks, scripts, visualizaciones y materiales de apoyo para explorar, limpiar y modelar datos. El objetivo es proporcionar pipelines claros y ejemplos prácticos que puedas adaptar a tus propios proyectos.

Características
---------------
- Notebooks Jupyter para limpieza, EDA y modelado.
- Pipelines reproducibles con scripts y requirements.
- Visualizaciones interactivas (Plotly / Altair) y estáticas (Matplotlib / Seaborn).
- Ejemplos de pre-procesado, ingeniería de variables y modelos básicos.

Requisitos
